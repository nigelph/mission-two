About Me

I am a recent AUT Graduate who completed a Bachelors in Computer and Information Science in early 2021. I am currently undertaking a Certificate in Technology Product Development Essentials with MissionReadyHQ which is a course designed to teach and learn all the tools needed to become a full stack web developer. This website started out as a "mission" to build a website using only HTML and CSS and has since been expanded on over time. More technologies were used progressively with the the aim to make use of tools taught in class by applying them. Since then, this website has become my personal page and ongoing project showcasing my growth and progression as a web developer.
